Verilerin Insertion Sort aşamaları: [22, 27, 16, 2, 18, 6] [22, 27, 16, 2, 18, 6] [16, 22, 27, 2, 18, 6] [2, 16, 22, 27, 18, 6] [2, 16, 18, 22, 27, 6] [2, 6, 16, 18, 22, 27]
Dizi Insertion Sort algoritması ile sıralandıktan sonra [2, 6, 16, 18, 22, 27] haline gelir. Insertion Sort'un Big-O gösterimi: O(n^2)
18 sayısının dizi içerisindeki pozisyonuna bakacak olursak, sıralandıktan sonra [2, 6, 16, 18, 22, 27] haline gelir. 18 sayısı, dizi içinde ortada bulunmaktadır. Bu durum, average(ortalama) case kapsamına girer.
Selection Sort'a göre ilk 4 adım: [2, 3, 5, 8, 7, 9, 4, 15, 6] [2, 3, 4, 8, 7, 9, 5, 15, 6] [2, 3, 4, 5, 7, 9, 8, 15, 6] [2, 3, 4, 5, 6, 9, 8, 15, 7] şeklinde ifade edilir.
